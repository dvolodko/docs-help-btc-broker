# Поповнення інвестиційного рахунку

Щоб мати можливість купувати Цінні Папери в Кабінеті Інвестора, потрібно поповнити ваш інвестиційний рахунок

В боковому меню Кабінета Інвестора перейдіть в розділ `Поповнення/Виведення` ➡️ `Поповнення/Виведення коштів`

Щоб поповнити інвестиційний рахунок за допомогою IBAN, скористайтесь блоком `Поповнення за реквізитами`.

### Копіюючи реквізити з веб-сторінки

:::tip

В блоці `Поповнення за реквізитами` реалізован механізм `Click to Copy`. Просто клікність по рядку і він скопіюється в буфер обміну

:::

### Надіславши собі електроного листа з реквізитами для поповнення

Якщо вам зручніше отримати реквізити на ваш ємейл, просто натисніть кнопку **`Надіслати на email`**

Протягом хвилини вам надійте лист з детальними реквізитами для поповнення інвестиційного рахунку
