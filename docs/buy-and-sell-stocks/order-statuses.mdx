# Перелік статусів заявок клієнта на купівлю/продаж фінансових інструментів в “Кабінеті Інвестора”

## Список назв статусів та короткий опис статусу

<Tabs
  defaultValue="submitted"
  values={[
    { label: 'Виставлена', value: 'submitted' },
    { label: 'Знята', value: 'withdrawn' },
    { label: 'Помилкова-Відхилена', value: 'rejected' },
    { label: 'Прийнята до виконання', value: 'accepted' },
    { label: 'Виконана', value: 'completed' },
  ]}>
  <TabItem value="submitted">
    Статус **"Виставлена"** надається заявці [“Лімітний
    ордер”](./placing-bids#лімітний-ордер), котру клієнт виставив у “Стакан
    котирувань” заповнивши поля “Моя кількість” та “Моя ціна” під таблицею
    котирувань. Відповідно під таку заявку блокуються активи клієнта (цінні
    папери/грошові кошти) до моменту її зняття або проведення угоди на біржі (у
    разі акцептування заявки частково чи повністю іншою стороною в “Стакані
    котирувань”)
  </TabItem>
  <TabItem value="withdrawn">
    Статус **"Знята"** надається заявці при натисканні клієнтом кнопки
    "Відмінити виставлені заявки” (знаходиться на сторінці “Стакану котирувань”
    над таблицею “Мої операції”. Зняти можливо тільки заявки зі статусом
    “Виставлена”).
  </TabItem>
  <TabItem value="rejected">
    Статус виникає у разі збою процедур проведення заявок на біржі та
    ініціюється “БТС Брокером”. Або клієнт просить “БТС Брокер” зняти
    “Виставлену” заявку, бо немає такої змоги з технічних причин. Відповідно
    результатом такого статусу є повернення заблокованих під заявку активів
    клієнта в “Кабінеті Інвестора” у початковий стан.
  </TabItem>
  <TabItem value="accepted">
    Статус **"Прийнята до виконання"** надається заявці коли вона акцептується з
    зустрічною заявкою в “Стакані котирувань”. При такому статусі заявка
    відправляється для проведення угоди на біржі упродовж двох робочих днів.
    Відмінити заявки з статусом "Прийнята до виконання" не можливо.
  </TabItem>
  <TabItem value="completed">
    Статус **"Виконана"** надається заявці після проведення угоди на біржі. При
    набутті даного статусу відбувається зарахування/списання цінних паперів та
    грошових коштів. Статус **“Виконана”** в “Кабінеті Інвестора” змінюється на
    наступний робочий день після проведення угоди на біржі. Про проведення угоди
    на біржі клієнту на імейл також приходить Виписка з рахунку в цінних паперах
    та Звіт про виконання замовлення
  </TabItem>
</Tabs>
