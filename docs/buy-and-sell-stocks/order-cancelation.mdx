# Відміна виставленої в “Трейдинговий стакан” лімітної заявки

Під “Стаканом котирувань” знаходиться таблиця **“Мої операції”** де можна побачити кнопку **“Відмінити виставленні заявки”**:

<div className="img-thumb">
  <img
    src={orderCancelation1}
    alt="таблиця Мої операції"
    className="img-preview"
  />
</div>

Натискання на дану кнопку призведе до зняття всіх виставлених інвестором заявок в “Стакані котирувань” відповідного фінансового інструменту:

<div className="img-thumb">
  <img
    src={orderCancelation2}
    alt="таблиця Мої операції"
    className="img-preview"
  />
</div>

Після чого статус заявки (заявок) зміниться на **“Знята”** та сам лімітний ордер зникне зі “Стакану котирувань”.

[`Повний перелік статусів заявок дивись ось тут`](./order-statuses)

:::warning

Важливе нагадування - Зняти можна тільки виставлені лімітні заявки ([`Лімітний ордер`](./placing-bids#лімітний-ордер)). Заявки зі статусом **“Прийнята до виконання”** - не відміняються!

:::
