# Регламент обробки платежів для Кабінету Інвестора

Години в рамках яких бухгалтерія “БТС Брокер” обробляє платежі на зарахування у робочі дні:\
**`Ранок`** \
З 8:00 до 9:00\
**`Обід`**\
з 12:00 до 13:00\
**`Вечір`**\
з 16:00 до 17:00

**Виводи грошей** з “Кабінету Інвестора” на свій банківський рахунок у робочі дні:\
Обробка заявок на вивід відбувається впродовж робочого дня до 15:30.

Виводи грошей по заявкам котрі подаються після 15:30, відбувається на наступний робочий день.

:::warning

Проведення транзакційних операцій без економічного сенсу таких операцій, а саме - поповнення свого брокерського рахунку з подальшим виведенням грошових коштів без проведення купівлі/продажу цінних паперів на дані суми - розглядаються окремо фінансовим моніторингом, `у такому випадку Регламент обробки платежів не діє`. Такі транзакції можуть бути заблоковані!

:::
